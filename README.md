# GitHub Zero

Simple GitHub app for iOS

## Getting Started

GitHub Zero uses [Cocoapods](https://cocoapods.org/)

``` 
git clone https://github.com/dkhamsing/github-zero.git
cd github-zero/Xcode
pod install
```
